Rest API App for Odoo v18
=========================

This module enables seamless communication with the Odoo database via RESTful API requests. It is designed for developers and integrators who need secure, configurable, and dynamic access to Odoo models and records.

Key Features
------------

- ✅ Dynamic API creation for any Odoo model or database table
- ✅ Access control for specific users or user groups
- ✅ Fine-grained permissions (Read / Write / Create / Unlink) per record and user
- ✅ Secure API endpoints for database operations
- ✅ Event logging to track API activity for audit and monitoring

Configuration Steps
-------------------

1. Install the **API APP** from your custom modules.
2. Create your desired API with one of the supported operations:
   - **READ** – Fetch data
   - **WRITE** – Update existing records
   - **CREATE** – Add new records
   - **UNLINK** – Delete records
3. Assign access rights by adding users under the **"User List"** tab on the API form.
4. View the API endpoint and usage instructions in the **"How to Use It"** tab.
5. Use the provided **cURL** request (or Postman) to test and interact with the API securely.
